# Shelby AI Evidence Vault

A public Shelby testnet ecosystem demo for verifiable AI evidence storage and read receipts.

## Goal

AI agents should be able to show what data they used, where it came from, and how it was retrieved.

This repository will build a polished web demo around evidence packs, dataset manifests, Shelby blob references, and read receipts.

## Current milestone

M0: public demo shell.

- Next.js / TypeScript / Tailwind scaffold
- Landing page
- Dashboard mock
- Upload mock
- Blob detail mock
- Read receipt mock
- Product guardrails for Copilot

## Hard scope

- Public ecosystem demo only
- No trading execution
- No private exchange APIs
- No marketplace or paid dataset trading
- No tokenomics modeling
- No wallet fund movement
- No production secrets
- No full AI agent framework
